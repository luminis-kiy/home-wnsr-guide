# home-wnsr-guide

一个用于归档和发布独立 HTML 页面的仓库。

## 项目简介

本仓库主要用于存放各类独立的 HTML 页面资源。这些页面彼此独立，不依赖特定的域名或网站环境，可被直接用于本地浏览或静态托管。仓库本身不提供任何在线服务或交互功能，仅作为页面资料的存档与分发点。

## 目录说明

```
home-wnsr-guide/
├── pages/        # 存放独立 HTML 页面文件
├── assets/       # 页面引用的公共资源（如 CSS、JS、图片等）
└── README.md     # 本文件
```

- `pages/`：包含所有归档的 HTML 页面，每个页面通常为一个独立的文件或子目录。
- `assets/`：存放页面可能共享的样式、脚本、字体、图像等静态资源，便于统一管理。

## 页面归档说明

- 所有页面均为纯前端实现，无后端依赖，可在浏览器中直接打开运行。
- 页面内容不涉及具体域名、商业推广或诱导访问行为。
- 归档的页面可能为工具、演示、文档、笔记或其他类型，具体用途见各页面文件内的说明。
- 仓库不保证页面内容长期有效或持续更新，仅作为历史存档使用。

## 维护说明

- 欢迎提交 Issue 或 Pull Request 来修正页面错误、改进样式或添加新的独立页面。
- 新增页面请放置在 `pages/` 目录下，并尽量保持资源引用路径的相对性。
- 如果页面需要外部资源，建议优先使用本地 `assets/` 目录下的文件，或引用稳定可靠的 CDN（不推荐用于敏感或长期存档场景）。
- 本仓库不承诺对页面内容进行审核或筛选，但会尽量保持存档的整洁与可用性。
